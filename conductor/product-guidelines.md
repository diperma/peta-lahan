# Product Guidelines

## Coding Standards
- **Surgical Changes**: Always prioritize minimal, precise changes to existing files.
- **ES Modules**: Use `import`/`export` syntax for all JS files.
- **Style Consistency**: Follow the CSS structure in `src/styles/`.
- **Error Handling**: Proxy server should robustly handle upstream timeouts and client disconnects.

## UI/UX Principles
- **Map Focus**: The map is the primary interface.
- **Responsiveness**: Ensure the app layout adapts to mobile devices.
- **Loading States**: Provide visual feedback for tile loading and API requests.
