# Tech Stack

## Frontend
- **Framework**: [Vite](https://vitejs.dev/) (Vanilla JS / TypeScript support ready)
- **Mapping Library**: [Leaflet](https://leafletjs.com/)
- **Styling**: Vanilla CSS (modular styles in `src/styles/`)
- **Deployment**: GitHub Pages

## Backend
- **Framework**: [Express.js](https://expressjs.com/)
- **Environment**: Node.js
- **Proxy Logic**: Custom streaming proxy using native Node `https` module.
- **Deployment**: [Vercel](https://vercel.com/) (configured in `vercel.json`)

## Development Tools
- **Concurrent Execution**: `concurrently` (to run both dev servers)
- **Version Control**: Git
